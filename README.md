# A/B Testing Analysis (Practice Project)

## *Overview*
Simple SQL project to analyze an A/B testing dataset and compare user behavior between **Ad group** and **PSA group**. The focus is on understanding conversion patterns using basic SQL queries.

---

## *Objective*
- Compare conversion rates between test groups  
- Analyze impact of ad exposure on conversions  
- Identify basic time-based patterns (day & hour)  
- Practice SQL aggregation and analysis

---

## *Key Results*

- **Conversion Rate**
  - Ad group: 2.55%  
  - PSA group: 1.79%  

- **Ad Exposure Impact**
  - Higher ads seen → higher conversion rate  
  - 100+ ads group shows highest conversion (~16.91%)

- **Time Patterns**
  - Best days: Monday & Tuesday  
  - Best hours: 15:00–20:00 range performs better  

---

## *Tech Stack*
- SQL (PostgreSQL)  
- Basic data aggregation and analysis  

---

## *Scope*
- Beginner-level SQL practice project  
- Focused on A/B testing logic and simple insights
